flex-wrap

The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

In other words, The flex-wrap property specifies whether the flexible items should wrap or not.

Note: If the elements are not flexible items, the flex-wrap property has no effect.

flex-wrap: nowrap

Default value. Specifies that the flexible items will not wrap.

flex-wrap: wrap
with flex-direction: column

Specifies that the flexible items will wrap if necessary.
The flex items break into multiple lines.
The cross-start is either equivalent to start or before depending flex-direction value and the cross-end is the opposite of the specified cross-start.
